# bear-blog-backup-template

A template for a backup of a bearblog.dev site

See https://github.com/RusticFlare/bear-blog-backup for an example usage.

To use this template:
1. Click "Use this template" and set set up your own repo  
2. Export your blog as a CSV or MD files and add them to the repo
3. Copy the contents of the parts of your blog that aren't exported into the files provided by this template
